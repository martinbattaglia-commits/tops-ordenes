Primeros pasos

¿Cómo usar este manual?

Seguí estos pasos para sacar el máximo provecho de la documentación.

1

Identificá tu rol

Cada módulo muestra qué roles tienen acceso. Filtrá por tu rol usando los botones del buscador para ver solo lo que te corresponde.

2

Seguí el orden recomendado

En la sección "Orden de lectura" encontrás la secuencia óptima según tu rol. Los módulos están priorizados para que aprendas lo más relevante primero.

3

Leé el manual del módulo

Hacé clic en "Ver manual" en la tarjeta de cada módulo. Cada manual explica en detalle las funcionalidades, campos y flujos de trabajo.

4

Marcá como leído

Usá el ícono ✓ en cada tarjeta para marcar el manual como leído. Tu progreso se guarda automáticamente en el navegador.

5

Consultá el mapa del sistema

El diagrama visual al final de esta página muestra cómo se conectan los módulos entre sí para entender el flujo global del sistema.

6

¿Dudas? Pedí ayuda

En la sección de ayuda encontrás el contacto de soporte interno y el canal para reportar errores o sugerir mejoras a la documentación.
